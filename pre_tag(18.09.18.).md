# html tag

## `<pre>`태그



```javascript
<!DOCTYPE HTML>
<html>
    
    <head>
        <meta charset='utf-8'>
        <title> 입력대로 보여주기 </title>
    </head>

    <body>
        입력한 대로 자유롭게 보여주는pre 태그입니다.

        <br>태그나 <p>태그가 아니더라도
        줄바꿈도 가능하며 태그도 그대로 표현할 수 있습니다.
    </body>

</html>
```

=>

입력한 대로 자유롭게 보여주는pre 태그입니다. `<br>`태그나 `<p>`태그가 아니더라도 줄바꿈도 가능하며 태그도 그대로 표현할 수 있습니다.
