### font-size



#### <u>***rem***</u>

html 태그의 폰트 크기에 따라서 **상대적**으로 크기가 결정 (권장)



***px***

폰트 크기가 고정된 값이기 때문에 사용자가 조정할 수 없다. (권장되지 않는다!)



***em***

부모 태그의 영향을 받는 상대적인 크기

### text-align



```html
<!doctype html>
<html>
    <head>
        <style>
            p {
                text-align: center;
                border:1rem solid gray;
            }
            /* left / right / center / justify */
            /* 왼쪽과 오른쪽이 공평하게 (양쪽이 균등하게 분배) */
        </style>
    </head>
    <body>
        <p>
            text
        </p>
    </body>
</html>
```
